# Rubix Lora Raw

## Running in development

- Use [`poetry`](https://github.com/python-poetry/poetry) to manage dependencies
- Simple script to install

    ```bash
    ./setup.sh
    ```

- Join `venv`

    ```bash
    poetry shell
    ```

- Build local binary

    ```bash
    poetry run pyinstaller run.py -n rubix-lora --clean --onefile --add-data VERSION:. --add-data config:config --hidden-import=src.utils.color_formatter
    ```

  The output is: `dist/rubix-lora`

## Docker build

### Build

```bash
./docker.sh
```

The output image is: `rubix-lora:dev`

### Run

```bash
docker volume create rubix-lora-data
docker run --rm -it -p 1919:1919 -v rubix-lora-data:/data --name rubix-lora rubix-lora:dev
```
